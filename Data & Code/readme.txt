REAL TIME GESTURE RECOGNITION 
------------------------------------------------
For this program to run, make sure you have installed opencv library 2.4 and python v2.7. 

Make sure the full path to the files 'left.jpg' and 'right.jpg' is specified in lines 208 and 209 of 'GestureRecognition.py'.

FACE DETECTION WITH GESTURE RECOGNITION 
------------------------------------------------
Make sure that the the compile python files 'video.pyc' and 'common.pyc' are in the same folder as the python script.

Also include the file 'haarcascade_frontalface_alt.xml' and ensure that the pathname of this file is correctly entered in Line 252 of 'FaceDetectionwithGestureRecognition.py'.
